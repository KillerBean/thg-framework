# THG install
passo a passo para fazer a instalacao do thg .

## arch linux
```
pacman -Syy python3 python3-pip
git clone https://github.com/darkcode357/thg-framework.git
cd thg-framework
pip3 install -r requirements.txt
```

## debian/ubuntu
```
apt install python3
apt install python3-pip
git clone https://github.com/darkcode357/thg-framework.git
cd thg-framework
pip3 install -r requirements.txt
```
