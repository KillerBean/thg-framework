# thg_concole install
passo a passo para fazer a instalacao do thg versao 1.0, voce precisa
insta

## arch linux
```
pacman -Syy python3 python3-pip
git clone https://github.com/darkcode357/thg-framework.git
cd thg-framework
python3 setup.py
```

## debian/ubuntu
```
apt update
apt install python3
apt install python3-pip
git clone https://github.com/darkcode357/thg-framework.git
cd thg-framework
python3 setup.py
```
